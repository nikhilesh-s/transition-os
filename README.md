# Transition OS

Transition OS is a student-built hub of templates, guides, and tools for managing classes, schedules, deadlines, and academic planning in one place.

## Quick start

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` ; start the local dev server
- `npm run build` ; build for production
- `npm run preview` ; preview the production build
- `npm run lint` ; run ESLint
- `npm run typecheck` ; run TypeScript checks

## Tech stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router

## Project structure

- `src/pages` ; route-level pages
- `src/components` ; shared UI components
- `src/index.css` ; global styles
- `src/main.tsx` ; app entry point

## Deploy

Build the production bundle with `npm run build` and serve the `dist` directory.
